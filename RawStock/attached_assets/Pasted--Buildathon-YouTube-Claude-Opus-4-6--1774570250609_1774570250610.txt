#### 必要なもの（無料〜低コスト）

### 3. Replitでメイラー立ち上げ手順（コピペでOK）

1. Replitで**新しいPythonプロジェクト**を作成  
2. **Secrets（環境変数）**に追加：
   - `ANTHROPIC_API_KEY` ← Claudeのキー
   - `SENDGRID_API_KEY` ← SendGridのキー
   - `SENDER_EMAIL` ← 検証済みのSendGridメールアドレス

3. `main.py` に以下のコードを貼り付け（最新Opus 4.6対応済み）：

```python
import os
from anthropic import Anthropic
import sendgrid
from sendgrid.helpers.mail import Mail, Email, To, Content

# 設定
anthropic = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])
sg = sendgrid.SendGridAPIClient(api_key=os.environ["SENDGRID_API_KEY"])
SENDER_EMAIL = os.environ["SENDER_EMAIL"]

# === ターゲットリスト（ここにCSV読み込みや手動追加）===
# 例: あなたのプロダクトが「AI営業自動化ツール」ならB2B SaaS創業者をターゲット
leads = [
    {"name": "山田太郎", "email": "target1@example.com", "company": "株式会社TechStart", "role": "CEO", "pain": "リード獲得コストが高い"},
    {"name": "鈴木花子", "email": "target2@example.com", "company": "株式会社GrowthLab", "role": "Marketing Manager", "pain": "メール開封率が低い"},
    # ← ここに数十〜数百件追加（CSVで一括読み込みも可）
]

def generate_personalized_email(lead):
    prompt = f"""
    あなたは世界最高のB2B営業コピーライターです。
    以下のターゲットに**超パーソナライズされた冷たい営業メール**を書いてください。
    - 名前: {lead['name']}
    - 会社: {lead['company']}
    - 役職: {lead['role']}
    - 痛み: {lead['pain']}
    - 私たちのプロダクト: Claude Opus 4.6を活用した完全自動営業ツール（ターゲット検索→メール生成→送信まで1クリック）

    条件:
    - 件名は30文字以内、クリックしたくなるもの
    - 本文は200-300文字、価値を先に提示
    - 必ず「返信不要・1クリックでデモ予約」リンクを入れる
    - 自然で押し売り感ゼロ
    - 日本語で
    """
    
    response = anthropic.messages.create(
        model="claude-3-opus-4-6-20250205",  # ← 最新Opus 4.6指定
        max_tokens=1000,
        temperature=0.7,
        messages=[{"role": "user", "content": prompt}]
    )
    return response.content[0].text

def send_email(to_email, subject, body):
    message = Mail(
        from_email=Email(SENDER_EMAIL, "あなたの名前 / プロダクト名"),
        to_emails=To(to_email),
        subject=subject,
        html_content=Content("text/html", body)
    )
    try:
        response = sg.client.mail.send.post(request_body=message.get())
        print(f"✅ 送信成功: {to_email} ({response.status_code})")
    except Exception as e:
        print(f"❌ 送信失敗: {to_email} → {e}")

# === 実行 ===
if __name__ == "__main__":
    print("🚀 Claude Opus 4.6 で営業自動化開始...")
    for lead in leads:
        email_content = generate_personalized_email(lead)
        # メール本文から件名と本文を分割（Claudeに「件名:xxx\n本文:yyy」の形式で出力させるようプロンプト調整可）
        # ここでは簡易的に全テキストを本文として扱い、件名は別途生成
        subject = f"{lead['name']}さん、貴社のリード獲得を1/10にしませんか？"
        send_email(lead["email"], subject, email_content)
```
